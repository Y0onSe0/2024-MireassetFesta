# 주식투자 입문자를 위한 맞춤형 뉴스 해설 서비스

> **2024 미래에셋증권 빅데이터 페스타**  
> Team 럭키통계쟈나 (양윤규 · 장윤서 · 현민영)

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **주제** | 주식투자 입문자를 위한 종목 지표 분석 및 맞춤형 뉴스 해설 서비스 |
| **데이터** | 네이버 금융 뉴스 크롤링 · 재무제표 · EV/EBITDA · 베타계수 |
| **AI 모델** | HyperCLOVA X (뉴스 요약 및 해설 생성) |
| **대회** | 2024 미래에셋증권 빅데이터 페스타 |

---

## 서비스 구성

```
사용자 투자 성향 테스트
  ↓ 종목 지표 분석 (EV/EBITDA · 베타계수 · 수익률)
  ↓ KMeans 클러스터링 → 투자 성향별 종목 분류
  ↓ 관련 뉴스 크롤링 및 전처리
  ↓ HyperCLOVA X → 종목별 맞춤 뉴스 해설 생성
  OUTPUT: 투자 성향에 맞는 종목 추천 + 뉴스 해설
```

---

## 노트북 구성

| 파일 | 내용 |
|------|------|
| `뉴스기사크롤링.ipynb` | 네이버 금융 뉴스 크롤링 (Selenium) |
| `뉴스전처리.ipynb` | 뉴스 텍스트 정제 및 전처리 |
| `뉴스종목추출.ipynb` | 뉴스에서 종목명 추출 |
| `뉴스해설추출.ipynb` | HyperCLOVA X 기반 뉴스 해설 생성 |
| `베타크롤링.ipynb` | 종목별 베타계수 크롤링 |
| `일별수익률표준편차계산.ipynb` | 수익률 변동성 계산 |
| `종목분류지표_클러스터링.ipynb` | KMeans 기반 종목 클러스터링 |
| `종목별_투자성향_뉴스해설.ipynb` | 투자 성향별 종목 뉴스 해설 통합 |

## 기술 스택

- **크롤링**: Selenium, BeautifulSoup
- **분석**: Python, pandas, numpy, scikit-learn
- **AI**: HyperCLOVA X (CLOVA Studio Fine-tuning)
- **클러스터링**: KMeans
